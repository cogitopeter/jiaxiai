# 嘉析 · 7维题目分析器

一套 Claude Code 自定义 Skill（Slash Command），覆盖小学至高中共9个学科。

拍一张题目照片，或直接打字输入题目，自动生成一份包含 **7个分析维度** 的精美 HTML 页面，并持续维护一个学习诊断仪表盘。

## 安装

将对应的 `.md` 文件复制到 `~/.claude/commands/` 目录，在 Claude Code 中输入命令名 + 题目图片即可。

首次使用时，Skill 会自动创建工作目录和索引文件，无需手动配置。

---

## 使用方式

```
/czshuxue [附上题目照片]
```

或先发送题目，再输入命令。支持触发词：`做这道题`、`分析这道题`、`这题怎么做`。

**注意：** 输入非对应学科题目时，Skill 会直接拒绝并提示，不生成任何文件。

## 7个分析维度

### 一、解题思路

不是标准答案，而是做题前的真实思考过程，以时间轴形式展示：

- **审题**：提取了哪些关键条件，第一反应是什么
- **尝试**：最初想用什么方法，走到哪一步发现走不通
- **突破**：怎么想到了正确的方法，关键洞察是什么
- **路径选择**：如果有多条路，为什么选了这条

语气像一个有经验的同学在旁边说"我是这么想的"，目标是激发思维模式，而非给出标准路径。

### 二、解题过程

至少提供 2 种解法，每种完整步骤。公式用 LaTeX 渲染（KaTeX），标注解法特点：通法 / 巧解 / 计算量对比。

### 三、知识点拆解

格式：`大模块 > 中模块 > 具体知识点`。覆盖该学科全部知识点，包括隐含的，并列出跨版本的不同叫法。

### 四、思维方式

逐步标注每个关键节点用了什么思维方法，并说明为什么在这里用。常见：数形结合、分类讨论、化归转化、整体思想、构造法、逆向思维……

### 五、易错点与陷阱

2-4 个常见错误，每个给出：❌ 错误做法 → ✅ 正确做法 → 原因分析。

### 六、变式题方向

2-3 个变式：改什么条件 → 变成什么题 → 核心区别。帮助举一反三。

### 七、难度定位

难度等级（基础/中档/中高档/压轴）、考试定位、建议用时、适合阶段。

## 学习诊断仪表盘

每分析一道题，索引页自动更新，包含三个视图：

- **题目列表**：卡片式排列，展开可看知识点和思维方式，一键跳转7个维度
- **知识模块**：按模块分组，快速定位某模块做过哪些题
- **学习诊断**：知识雷达图、难度分布柱状图、思维方式词云、模块掌握度排名、做题热力图

做得越多，诊断越准。

## 技术细节

- 公式渲染：KaTeX 0.16.9 CDN
- 题目页面：纯静态 HTML，白色主题，支持打印
- 索引页面：纯静态 HTML + 内联 JavaScript，深色主题，SVG 可视化
- 无需后端、无需数据库，所有数据存在本地 HTML 文件中，可离线浏览
